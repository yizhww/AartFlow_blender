# 项目结构

## 📁 项目结构

```
AartFlow_blender/
├── README.md                    # 项目说明文档
├── LICENSE                      # GPL-3.0 许可证
├── .gitignore                   # Git 忽略规则
├── texture/                     # 资源文件
│   └── 1.png                   # AartFlow Logo
├── docs/                       # 📚 文档中心
│   ├── README.md               # 文档索引
│   ├── installation.md         # 安装指南
│   ├── features.md             # 功能特性
│   ├── quick-start.md          # 快速开始
│   ├── tutorials.md            # 使用教程
│   ├── use-cases.md            # 适用场景
│   ├── development.md          # 开发指南
│   ├── module-development.md   # 模块开发
│   ├── api-reference.md        # API 参考
│   ├── packaging.md            # 打包发布
│   ├── versioning.md           # 版本管理
│   ├── contributing-and-support.md  # 贡献与支持
│   ├── license.md              # 许可证信息
│   └── project-structure.md    # 项目结构
└── AartFlow/                   # 插件主目录
    ├── __init__.py             # 插件入口点
    ├── AARTFLOW_integration.py # 核心集成器
    ├── manifest.json           # 插件清单文件
    ├── README.md               # 插件详细说明
    └── scripts/                # 业务脚本模块目录
        ├── artRenderFront.py  # 渲染前端模块
        ├── artRenderBack.py   # 渲染后端模块
        ├── objectMeasure.py    # 对象测量工具
        ├── dataPlotting.py     # 数据可视化
        ├── standardView.py     # 标准视图管理
        ├── skylightsManage.py  # 天窗管理
        ├── skpDrag.py         # SKP 拖拽功能
        └── openCmd.py         # 命令行工具
```
